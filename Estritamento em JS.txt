A boa prática do estritamento no JS 

No JavaScript tem uma função diretiva chamada "use strict" que basicamente obriga o seu código js, a seguir algumas boas práticas que são:
Não é permitido criar variáveis globais implicitamente. Todas as variáveis globais devem ser declaradas usando a palavra-chave var, let ou const.
Não é permitido usar palavras-chave reservadas como nomes de variáveis ou funções.
Não é permitido modificar a propriedade [[Prototype]] de um objeto.
Não é permitido usar a função delete em variáveis, funções ou parâmetros.
Não é permitido usar a declaração with.

Para usar você deve escrever no início do código
"use strict";
que já vai estar funcionado
